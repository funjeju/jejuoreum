# 12. Print Passport

> 본 문서는 인쇄물 패스포트의 디자인·구성·인쇄 사양을 정의한다.
> 인쇄물은 본 프로젝트의 메인 상품이자 1차 수익원이며, 디지털 웹앱과 동일한 비주얼 언어를 공유한다.

---

## 0. 인쇄물의 위상

### 0.1 인쇄물이 메인 상품인 이유

본 프로젝트는 디지털 웹앱이 함께하지만, **인쇄물 패스포트가 메인 상품**이다.

- **수익 구조의 1차 수익원**: 패스포트 판매가 핵심 매출
- **수집의 본질**: 손에 쥐는 도장의 손맛은 디지털이 대체 못 함
- **차별화 자산**: 디지털만으로는 만들기 어려운 해자
- **선물 가능**: 실물이 있어야 선물·소장 가능

### 0.2 인쇄물과 디지털의 역할 분담

| 영역 | 인쇄물 | 디지털 |
|------|--------|--------|
| 일러스트 카드 | **메인 비주얼** | 동일 일러스트 사용 |
| 오름 인증 | 보조 (시각적 도장 흔적) | **메인 (GPS 기반)** |
| 제휴 상권 인증 | **메인 (아날로그 도장)** | 미연동 |
| 정보량 | 핵심만 | 깊이 있게 |
| 갱신 | 불가 (고정) | 자유롭게 |
| 미감 | 손에 쥐는 소장품 | 살아있는 카드 |

### 0.3 디자인 원칙

본 인쇄물 디자인은 다음 원칙을 따른다.

1. **소장 가치**: 다 모은 후에도 책장에 꽂아두고 싶은 미감
2. **여행에 들고 다닐 수 있는 크기·내구성**: 가방에 들어가고 비에 약하지 않게
3. **자기 기록의 즐거움**: 도장 자리, 메모 자리가 충분히 매력적이게
4. **디지털과 시각 통일**: 같은 일러스트, 같은 색감, 같은 폰트

## 1. 패스포트 사양 개요

### 1.1 책 사양

| 항목 | 사양 |
|------|------|
| **판형** | A6 변형 (140mm × 100mm) |
| **두께** | 약 12~15mm (110~130페이지) |
| **표지** | 양장 (하드커버) 또는 PUR 무선철 |
| **종이 (본문)** | 미색 모조지 100g 또는 색감 좋은 라이트 크라프트 |
| **종이 (카드면)** | 매트 코팅 200g (도장 잘 묻고 일러스트 잘 살게) |
| **페이지 수** | 약 130페이지 (앞부분 가이드 + 100선 + 뒷부분 부록) |
| **제본** | 펼치기 좋은 누드 양장 검토 (사용성) |
| **모서리** | 둥글림 (휴대 시 손상 방지) |

**판형 선정 근거**:
- 너무 작으면 도장 찍기 어렵고 일러스트 미감 떨어짐
- 너무 크면 등산·여행 가방에 부담
- A6 변형(140×100)은 손에 잡히는 가장 큰 크기 + 가독성 양립

### 1.2 가격 (참고)

- 권장 소비자가: 25,000~35,000원
- 첫 인쇄 1,000부 기준 단가: 약 5,000~8,000원 (양장 기준)
- 마진 여유: 약 60~75%

상세 비즈니스 모델은 페이즈 0 운영 계획에서 결정.

### 1.3 초도 인쇄 부수

- 1차 인쇄: 1,000~2,000부
- 시장 반응 보고 2차 추가
- 외부인쇄소 발주 (제주 또는 서울 거래처)

## 2. 패스포트 구성 (목차)

### 2.1 전체 구조 (약 130페이지)

```
[1. 표지]
   - 앞표지: 일러스트 + 제목 + 부제
   - 뒷표지: 짧은 카피 + 가격 표시

[2. 면지·속표지] (4페이지)
   - 면지 (디자인 패턴)
   - 속표지

[3. 인사말 + 사용 안내] (4~6페이지)
   - 환영 인사
   - 패스포트 사용법 (그림 인포그래픽)
   - QR 안내 (단일 QR 위치)

[4. 제주 오름 입문] (4~6페이지)
   - 오름이란?
   - 제주 오름 분포 지도 (양면 펼침)
   - 베스트 시즌 가이드
   - 탐방 매너

[5. 100선 카드 영역] (100페이지)
   - 비기너 30개 (페이지 1~30)
   - 익스플로러 70개 (페이지 31~100)
   - 각 오름 1페이지

[6. 부록] (10~15페이지)
   - 제휴 상권 리스트
   - 추천 코스 5~10개
   - 마지막 페이지: 종합 도감 완성 인증
   - 필기·메모 페이지 (3~5페이지)

[7. 뒷면지·뒷표지]
```

### 2.2 페이지 분배 (정확)

| 영역 | 페이지 |
|------|--------|
| 표지 | 2 |
| 면지 (앞) | 2 |
| 속표지 | 1 |
| 빈 페이지 | 1 |
| 인사말 | 2 |
| 사용 안내 | 2~4 |
| 제주 오름 입문 | 4~6 |
| 분포 지도 (양면) | 2 |
| 100선 카드 | 100 |
| 제휴 상권 리스트 | 4 |
| 추천 코스 | 4~6 |
| 종합 인증 | 2 |
| 메모 페이지 | 3~5 |
| 면지 (뒤) | 2 |
| **합계** | **약 130** |

## 4. 사용 안내 페이지

### 4.1 인포그래픽 4컷

```
[1] 인쇄물 + 디지털 = 함께
   ─ 그림: 책과 폰이 나란히
   ─ "두 가지 방법으로 도감을 채워요"

[2] 오름 인증
   ─ 그림: GPS 폰 아이콘
   ─ "QR 스캔 → 가까운 오름 자동 인식"
   ─ "발견 시 카드가 컬러로 변해요"

[3] 제휴 상권 도장
   ─ 그림: 도장 찍는 손
   ─ "근처 제휴 카페/식당에서 도장 받기"
   ─ "스티커 함께 모으는 재미"

[4] 함께 즐기기
   ─ 그림: 코스 지도
   ─ "위시리스트로 다음 코스 짜기"
   ─ "친구와 도감 공유"
```

### 4.2 다국어 안내 (페이즈 4 검토)

페이즈 1 출시 시점에는 **한국어 인쇄물만**.

페이즈 2~3에서 영어 인쇄물 별도 발행 검토:
- 영어 인쇄물은 외국 관광객 대상
- 동일 100개 카드 + 영어 본문
- 출시 시점에는 안 나옴

## 5. 100선 카드 페이지 디자인

가장 중요한 페이지. 100페이지 모두 동일한 양식.

### 5.1 페이지 레이아웃

```
┌────────────────────────────────────┐
│  001                  비기너        │ ← 상단 헤더
│                                    │
│  새별오름                           │ ← 오름명 (큰 폰트)
│  Saebyeol Oreum                    │ ← 영문 (작게)
│                                    │
│  ┌──────────────────────────────┐  │
│  │                              │  │
│  │                              │  │
│  │     [카드 일러스트]            │  │
│  │      (전체 폭의 80%)           │  │
│  │                              │  │
│  │                              │  │
│  └──────────────────────────────┘  │
│                                    │
│  📍 제주시 애월읍                   │ ← 한 줄 요약
│  ⛰️ 표고 519m · 비고 119m          │
│  ⏱️ 탐방 30분 · 난이도 ★☆☆☆☆      │
│                                    │
│  ┌─────────────┐  ┌────────────┐  │
│  │             │  │            │  │
│  │   [도장 자리]│  │  [메모 자리] │  │
│  │   원형 35mm  │  │   (날짜 +    │  │
│  │             │  │    한 줄)    │  │
│  └─────────────┘  └────────────┘  │
│                                    │
│  ▶ 가는 길 팁: 셔틀버스 운영 중     │
└────────────────────────────────────┘
```

### 5.2 페이지별 상세 명세

**상단 헤더 (10mm 높이)**
- 좌: 카드 번호 (001~100)
- 우: 영역 라벨 (비기너 / 익스플로러)
- 폰트: --text-tiny (11px), 보통 weight
- 색상: --neutral-700

**오름 이름 영역 (15mm 높이)**
- 한글 이름: 24pt, weight 700, --neutral-900
- 영문 이름: 12pt, weight 400, --neutral-500
- 좌측 정렬

**카드 일러스트 영역 (60mm 높이)**
- 가로 폭: 페이지 폭의 80% (좌우 14mm 여백)
- 세로 폭: 60mm
- 비율: 4:3
- 모서리: 라운드 6mm
- 디지털과 동일한 일러스트
- CMYK 변환 후 톤 보정

**메타 정보 영역 (15mm 높이)**
- 3줄 (위치 / 표고+비고 / 탐방시간+난이도)
- 폰트: 11pt, weight 400
- 이모지 또는 아이콘 사용

**도장·메모 영역 (40mm 높이)**

도장 자리:
- 원형 35mm × 35mm
- 점선 테두리 (스탬프 위치 가이드)
- 안에 옅은 가이드 텍스트 ("STAMP")
- 좌측 배치

메모 자리:
- 가로선 3~4줄
- 날짜 적는 자리 (위)
- 우측 배치

**팁 영역 (10mm 높이)**
- 한 줄 팁 (가는 길, 주차, 화장실 등 핵심 정보)
- 폰트: 11pt, weight 500
- 색상: --brand-green-800

### 5.3 100페이지 일관성

모든 100페이지가 위 레이아웃으로 통일.
페이지마다 다른 것은:
- 카드 번호
- 영역 라벨 (비기너/익스플로러)
- 오름 이름
- 일러스트
- 메타 정보 텍스트
- 팁 텍스트

이것 외 모든 시각 요소(여백, 폰트, 색)는 100% 동일.

### 5.4 비기너 vs 익스플로러 시각 차이

미세한 차이로 영역 구분:
- **비기너 페이지**: 상단 헤더에 옅은 그린 라인
- **익스플로러 페이지**: 상단 헤더에 짙은 그린 라인
- 두 영역 사이에 **간지 페이지** 1장 ("Explorer 영역으로 들어갑니다")

## 7. 제휴 상권 영역

### 7.1 상권 리스트 페이지 (4페이지)

각 페이지마다:

```
┌────────────────────────────────────┐
│  제휴 상권                           │
│  ── Partner Merchants ──            │
├────────────────────────────────────┤
│                                    │
│  📍 카페 오롯                       │
│  새별오름 인근 · 카페                │
│  ☎️ XXX-XXXX-XXXX                   │
│                                    │
│  ─────────────────────────         │
│                                    │
│  📍 다랑쉬 식당                      │
│  다랑쉬오름 인근 · 식당              │
│  ...                               │
│                                    │
└────────────────────────────────────┘
```

각 상권에:
- 이름, 위치, 종류, 연락처
- 도장 자리 1개 (원형 25mm)

### 7.2 도장 시스템 (아날로그)

**중요**: 제휴 상권 도장은 디지털과 미연동.

- 점주가 본인 도장 보유
- 패스포트 지참 시 점주가 직접 찍어줌
- 한 상권에서 여러 번 받아도 도장 1번만 (의미)

### 7.3 도장 디자인 가이드

상권 입점 시 운영팀이 제공:
- 통일된 디자인 템플릿
- 상권명 + 일러스트 (오름 또는 상권 마스코트)
- 직경 25mm 원형
- 옵션: 오일 스탬프 또는 일반 도장 잉크

운영 측이 도장 발주·배송 책임 (`13_partner_merchants.md` 참조).

## 11. 인쇄 사양 (제작 시 명세)

### 11.1 색상

- **컬러 모드**: CMYK
- **사용 잉크**: CMYK 4도 인쇄
- **별색**: 사용 안 함 (비용 효율)
- **컬러 프로파일**: ISO Coated v2 (FOGRA39) 또는 동등

### 11.2 색상 변환 시 점검

`16_design_system.md`의 RGB 토큰을 CMYK로 변환할 때:

| 토큰 | RGB Hex | CMYK 변환 (참고) |
|------|---------|----------------|
| --brand-green-800 | #1A4D2E | C84 M40 Y100 K40 |
| --brand-green-100 | #D4E8DC | C20 M0 Y15 K5 |
| --neutral-900 | #1A1A1A | C0 M0 Y0 K100 |
| --neutral-50 | #FAFAFA | C0 M0 Y0 K2 |

**중요**: 인쇄 시 약간 어둡게 나옴. 시제품(校了) 한 부 받아서 색감 점검 필수.

### 11.3 출력 사양

- **해상도**: 300 DPI 이상
- **재단선**: 3mm 여유
- **재단 마크**: 표시 필수
- **블리드**: 3mm
- **안전 여백**: 5mm
- **페이지 번호**: 본문 페이지에 작게 (--neutral-500, 9pt)

### 11.4 일러스트 출력 준비

일러스트 작업 시:
- 원본: RGB로 작업
- CMYK 변환은 인쇄 직전 (인쇄소가 처리하거나 사전 변환)
- 사이즈: 인쇄 사이즈 × 300 DPI = 약 1500×1100 픽셀 (카드 한 장당)
- 포맷: TIFF (인쇄용) 또는 PDF/X-1a

### 11.5 종이 샘플 사전 검토

- 본문 종이 후보 3~4종 → 종이 샘플 받아 인쇄 테스트
- 매트 코팅 표지 vs 무광 vs 천 양장 비교
- 도장 시 잉크 번지는지 테스트 (필수)

## 12. 인쇄소 협업

### 12.1 인쇄소 선정 기준

- 양장 제본 가능
- 소량 인쇄 가능 (1,000부)
- CMYK 인쇄 품질 검증
- 한글 타이포 작업 경험

후보:
- 서울: 알래스카 인디고, 코랄프린트 등
- 제주: 제주 현지 인쇄소 (배송비 절감)

### 12.2 발주 프로세스

```
[원고 완성]
    ↓
[인쇄소 견적 (2~3곳 비교)]
    ↓
[교정쇄 (시제품) 요청]
    ↓
[색감 점검 + 도장 잉크 테스트]
    ↓
[수정사항 반영]
    ↓
[본 인쇄 (1,000부)]
    ↓
[검수 + 입고]
```

### 12.3 일정 추정

- 디자인 완료부터 입고까지: 약 2~3개월
- 그 중 인쇄·제본 자체는 약 3~4주
- 시제품 점검 1~2주
- 페이즈 0 일정에 반드시 반영

## 13. 패키징·유통

### 13.1 패키징

- 비닐 슈링크 (개별 포장)
- 옵션: 종이 슬리브 (선물용)
- 라벨: 제목 + 가격 + ISBN (있다면)

### 13.2 유통 채널

- **공식 웹사이트**: 1차 판매처 (수익 최대)
- **제주 공항·기념품점**: 위탁 판매
- **온라인 서점**: 교보문고, 알라딘 등 (페이즈 2)
- **제주 주요 카페·게스트하우스**: 제휴 상권 통한 위탁
- **여행사 패키지**: 페이즈 3 검토

### 13.3 ISBN 등록

- 도서로 분류하여 ISBN 발급 검토
- 출판사 등록 또는 공동 발행 협력 검토
- ISBN 있으면 서점 진입 용이

### 13.4 재고 관리

- 첫 1,000부 → 2~3개월 판매 추적
- 판매 속도 보고 2차 인쇄 결정
- 시즌 한정판 (한정 표지 색·일러스트) 페이즈 3 검토

## 14. 인쇄물 한정판 전략 (페이즈 3+)

### 14.1 시즌 한정 표지

- 봄/여름/가을/겨울 4종 표지
- 본문은 동일, 표지만 변경
- 컬렉터블 가치

### 14.2 한정 일러스트 카드

- 100선에 추가되는 새 오름 (마스터 영역) 카드 별도 인쇄
- 기존 패스포트 + 추가 카드팩 형태
- 디지털 마스터 영역과 연동

### 14.3 챌린지 완료 굿즈

- 비기너 마스터 → 한정 스티커
- 100선 완주 → 한정 책갈피, 종합 인증 도장
- 제주 마스터 → 한정 굿즈 패키지

## 15. 디지털 연동 포인트

인쇄물에서 디지털로 자연스럽게 연결되는 지점.

### 15.1 QR

- 표지 안쪽 단일 QR → 웹앱 진입
- 100개 카드에는 별도 QR 없음 (GPS 자동 매칭)

### 15.2 종합 인증 페이지

- 100선 완주 시 디지털 "제주 마스터" 메타 배지 자동 트리거
- 인쇄물 종합 도장 + 디지털 배지 둘 다 받는 구조

### 15.3 시즌 한정판

- 한정판 구매 시 디지털 시즌 카드 잠금 해제 코드 동봉
- 코드 입력 → 디지털 한정 카드 추가

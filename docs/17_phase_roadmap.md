# 17. Phase Roadmap

> 본 문서는 제주 오름 패스포트 프로젝트의 페이즈별 상세 로드맵을 정의한다.
> 각 페이즈의 작업 목록, 의존성, 마일스톤, 일정 추정, 리스크를 모두 포함한다.
> 본 문서는 프로젝트 진행 시 **단일 진실 공급원(SSOT)** 역할을 한다.

---

## 0. 페이즈 정의 재확인

### 0.1 핵심 원칙

`core.md`와 이전 결정에서 확정:

**페이즈는 기술 의존성 순서다. 기능 분량 조절 도구가 아니다.**

- 출시 시점에 들어가야 할 기능은 다 들어간다
- 페이즈 분리 기준: 기술적 선후 관계
- 페이즈 후행 작업: 사용자 베이스 임계점 또는 콘텐츠 누적 시점 의존

### 0.2 페이즈 구조

| 페이즈 | 기간 | 목표 |
|--------|------|------|
| **페이즈 0** | 출시 전 (3~5개월) | 인프라 + 콘텐츠 + 인쇄물 준비 |
| **페이즈 1** | 출시일 + 안정화 1~2주 | 핵심 기능으로 출시 |
| **페이즈 2** | 출시 후 3~6개월 | AR + 피드 + 챌린지 본격화 |
| **페이즈 3** | 출시 후 6~12개월 | 굿즈·수익 다각화 + 마스터 영역 |
| **페이즈 4** | 출시 후 1년+ | 자산 활용 + 글로벌 + 팀 시스템 |


### 1.4 Critical Path (최장 경로)

가장 길어 출시 일정을 결정하는 작업들:

1. **카드 일러스트 100장**: 6~8주 (병렬 작업 어려움)
2. **SEO 콘텐츠 100선 100개 깊이**: 8~12주 (병렬 가능하나 품질 위해 단일 작성자 권장)
3. **인쇄물 인쇄·제본**: 6주 (디자인 완료 후 인쇄소 작업)
4. **제휴 상권 80~150곳 모집**: 6~10주

병렬 처리 가능하지만 누적 시간 가장 긴 항목들.

### 1.5 페이즈 0 총 추정 기간

**최소 3개월, 권장 4~5개월.**

분기별 보면:
- M1 (1개월): 인프라 + DB 입력 + 디자인 시작
- M2 (2개월): 일러스트 진행 + SEO 콘텐츠 50% + 인쇄물 디자인 + 제휴 컨택

- M4 (4개월): 인쇄물 입고 + 도장 발송 + 통합 테스트 + 베타
- M5 (5개월): 안정화 + 출시 준비

## 2. 페이즈 1: 출시

### 2.1 목표

**핵심 사용자 경험을 안정적으로 제공**한다. 출시 첫 1~2주는 안정화에 집중.

### 2.2 출시 시점에 작동해야 할 기능

#### 인증·도감
- ✅ QR 스캔 → GPS 3단계 매칭 → 인증
- ✅ 발견 모먼트 (흑백→컬러)
- ✅ 마이페이지 도감 (4개 탭)
- ✅ 진척도 시각화

#### 콘텐츠
- ✅ 카드 페이지 (sticky 카드 + 11섹션)
- ✅ SEO 페이지 360개 (100선 100 + 비100선 50~100 + 발행 안 된 것들)
- ✅ 검색엔진 등록 완료

#### 게이미피케이션
- ✅ 24+ 배지 작동 (자동 트리거)
- ✅ 시즌 챌린지 5~10개 (출시 시점)
- ✅ 리듬 시스템 (이번 달 발견, 협력감 표시)

#### 차별화
- ✅ 오름 MBTI 시스템 (퀴즈 + 추천)
- ✅ 위시리스트 + 동선 설계 도구
- ✅ 사진 업로드 + AI 자동 분류
- ✅ 코멘트 + AI 분석

#### 인프라
- ✅ 카카오/네이버 OAuth
- ✅ PWA (홈 화면 추가, 오프라인)
- ✅ 다국어 인프라 (한국어 100% 콘텐츠)
- ✅ 관리자 백오피스 (최소 기능)

#### 인쇄물·제휴
- ✅ 인쇄물 1,000부 입고
- ✅ 제휴 상권 80~150곳 활성
- ✅ 도장 발송 완료
